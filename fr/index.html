Ma cuisine

Mon Univers

  • La décoration n’a pas de code.

  • La liberté est de mise à table.

3 raisons pour cuisiner

Pourquoi je cuisine?

- Pour manger sain.
- Pour réaliser mes propres recettes.

Style de cuisine?

- Une cuisine simple, amusante, savoureuse, avec peu d’ingrédients, se basant plus sur les légumes que les viandes, avec un temps de cuisson réduit.

Pourquoi je partage?

- Pour témoigner que nous pouvons manger sain, bon marché et dans l’élégance malgré le manque de temps.
- Pour vous donner l’envie de cuisiner et non vous apprendre à cuisiner.

Mes ingrédients de base

Les légumes, les fruits, les graines, les plantes aromatiques et les épices.

Mes ingrédients les + appréciés

Les poissons blancs, les fromages, les légumineuses et les fruits secs.

Mes ingrédients les + consommés

L'ail, le gingembre frais, le citron et le persil.

Mes ingrédients

Le sel, le sucre et les produits industrialisés.
